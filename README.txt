# Controle de Despesas

## Objetivo
Este projeto é um script em Python que ajuda o usuário a calcular despesas mensais com base em entradas fornecidas, gerando um relatório simples e organizado.

## Funcionalidades
1. Entrada de categorias de despesas e seus valores.
2. Cálculo automático de totais.
3. Relatório organizado com resumo das despesas.

## Cronograma
- Dia 1: Configuração do repositório e escrita do README.md.
- Dia 2: Implementação da lógica de entrada de dados e cálculos.
- Dia 3: Geração de relatórios.
- Dia 4: Testes e finalização.
